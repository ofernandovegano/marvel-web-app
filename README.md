# Marvel-web-app

Web application based on a marvel API that is useful to search and learn about marvel comics and characters from an official source.

## Access

Access through the link [https://marvelous-comic-app.herokuapp.com/](https://marvelous-comic-app.herokuapp.com/).

## Usage

First step: Make an account
Second Step: Choose What you want to search: comics or characters.
Third step: Search using the search bar or navigate through the page to find the comics or the caracthers.

*You can save the comics and characters as favorites. They will be accessible through the link in the navbar.

Enjoy it!
